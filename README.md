# 图片压缩工具

一个免费、易用的在线图片压缩工具，支持批量上传和压缩，适用于各种设备。

## 功能特性

### 📱 移动端友好
- 支持移动端相册勾选功能
- 响应式设计，适配各种屏幕尺寸
- 触摸友好的界面设计

### 📁 批量处理
- 支持同时上传和压缩多张图片
- 压缩数量不受限制
- 支持拖拽上传

### 🎨 灵活设置
- 可调节压缩质量（10%-100%）
- 支持多种输出格式（JPEG、PNG、WebP）
- 实时预览压缩效果

### ⚡ 高效压缩
- 本地压缩，保护隐私
- 快速处理，无需等待
- 清晰的压缩前后对比

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（响应式布局）
- **JavaScript** - 核心功能实现
- **Canvas API** - 图片压缩处理

## 使用方法

### 基本使用

1. **上传图片**
   - 点击上传区域或拖拽图片到上传区域
   - 在移动端可直接从相册选择多张图片

2. **调整设置**
   - 使用滑块调整压缩质量
   - 选择所需的输出格式

3. **开始压缩**
   - 点击"开始压缩"按钮
   - 等待压缩完成（显示进度）

4. **下载图片**
   - 点击单张图片的"下载"按钮下载单个文件
   - 点击"下载全部"按钮批量下载所有压缩后的图片

### 移动端使用

- 在手机浏览器中打开网页
- 点击上传区域，选择"相册"
- 勾选多张图片进行上传
- 压缩完成后下载到手机相册

## 核心功能说明

### 图片压缩原理

使用HTML5 Canvas API将图片绘制到画布上，然后通过 `toBlob()` 方法将画布内容转换为指定格式和质量的图片文件。

### 支持的图片格式

- **输入格式**：JPEG, PNG, WebP, GIF, BMP
- **输出格式**：JPEG, PNG, WebP

### 压缩质量说明

- **高质量**（80%-100%）：适合需要保持高质量的图片
- **中等质量**（50%-80%）：平衡质量和文件大小，推荐使用
- **低质量**（10%-50%）：适合对质量要求不高的场景

## 性能优化

- 本地压缩，无需服务器资源
- 使用Web Workers处理大量图片（规划中）
- 图片懒加载和渐进式加载
- 优化的Canvas压缩算法

## 隐私保护

- 所有图片处理均在本地浏览器完成
- 不会将图片上传到任何服务器
- 刷新页面后，所有数据将被清除
